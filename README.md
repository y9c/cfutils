# cfutils

**Chromatogram File Utils**

for sanger sequencing data visualizing, alignment, mutation calling, and trimming etc.


## How to install?

install dependance

```bash
make init
```

do unittest

```bash
make test
```

## How to use?
 
```python
import cfutils as cf

```
