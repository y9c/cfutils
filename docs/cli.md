# Command Line Interface (CLI)

## Detect and visualize mutations in one step

```bash
cfutils mut --query ./data/B5-M13R_B07.ab1 --subject ./data/ref.fa --outdir ./data/ --plot
```

## For help on CLI options

```bash
cfutils mut --help
```

## Running Tests

To run all tests:

```bash
make test
```

To clean up build and cache files:

```bash
make clean
```
